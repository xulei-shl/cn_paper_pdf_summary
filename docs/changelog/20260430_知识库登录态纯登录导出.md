- **日期**: 2026-04-30

## 1. 变更摘要
为知识库上传脚本新增 `--login-only` 模式，用于不传入 Markdown 文件时执行登录检测、等待手动登录并导出 Playwright 登录状态。
将上传脚本底层浏览器从 Chromium 持久化上下文切换为 Camoufox 持久化上下文，统一与仓库内其他反检测脚本的浏览器方案。
将 `--login-only` 的登录成功判定从页面元素自动检测调整为终端人工确认，避免登录后页面状态不稳定导致无法继续导出。

## 2. 文件清单
